☁️ 設定 Google Drive 同步
只需做一次，之後自動同步。全程免費。
📋 步驟一：取得 Google Client ID
  1. 前往 console.cloud.google.com（免費，需 Google 帳號）
  2. 建立新專案（隨意命名，如「阿丹管理」）
  3. 左側選「API 和服務」→「已啟用的 API」→ 啟用「Google Drive API」
  4. 左側選「憑證」→「建立憑證」→「OAuth 2.0 用戶端 ID」
  5. 應用程式類型選「網頁應用程式」
  6. 「已授權的 JavaScript 來源」填入您的網站網址（例：https://adan.netlify.app）
    本機測試填：http://localhost 和 http://127.0.0.1
  7. 複製產生的「用戶端 ID」（格式：xxxx.apps.googleusercontent.com）
📝 步驟二：填入 Client ID
✅ 完成後
• 每次儲存資料自動備份到您的 Google Drive
• 換電腦只需重新登入 Google，點「從雲端還原」
• 備份檔名：阿丹管理系統備份.json
💡 替代方案：如果設定太複雜，可以用「下載備份檔」功能，每週下載一次 JSON 備份檔存在 Google Drive 裡，換電腦時手動上傳還原。雖然需要手動操作，但完全不需要設定任何東西。
